A social media website across different platforms

Little sheep is a social media site for knitters. It allows users to view otherâ€™s creation, save tutorials, join events and start conversations. The main challenge of this assignment is having to create a profile page across three screen sizes with consistent style. The develop process starts with hand drawn wire framing, then later gray tone wire framing, next a high fidelity wire frame, finally touch is connect the flow using Axure.

Monitor: 1680 x 1050 pixel resolution
Tablet: 1024 x 768 pixel resolution
Smartphone: 640 x 1136 pixel resolution

Process

The end goal is to develop a cohesive digital interface for a functional social media website across platform. I started research and identified the target user. I looked at the existing applications and websites. From there I researched on users' feedback to have a clear idea of their needs. I started with rough drafts for fast iterations, then used illustrator to construct a digital grayscale wireframe interface, based on the wireframe I refined and improved to a high fidelity wireframes, finally a interactive scenario prototype.
