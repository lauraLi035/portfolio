Key points

HTML Layout Elements

Each HTML file must starts with <html> and end with </html> tag
<head> - defines the title of the document in <title>
<body> - contains the information you want to display
< > is a open tag and </> is a closed tag

basic HTML < >

< h1 > element defines a large
< h2 > element defines a smaller heading
you try from < h1 > to < h6 >
< p >element defines a paragraph
< br >indicates a line break, white space don't work in HTML

Images

Images are defined with the < img >
with provided source file (URL)

< img src="pokemon.jpg">

hyperlinks

link page to page, 2type of URL, think URL as address

  1. destination address ie. http://www.google.com
  2. relative URL, think as you are at home and give directions base on your location
    • name the file to call open
    • ../ to go to last directory
file management

  1. save same type of file together. IE, images in one file, HTML in one files
  2. save a group of files together IE, save all the files from today's in week1
styling in HTML

  1. change font color
    <p style="color:red;">
  2. change background color
    • as a whole
      < body style = "background-color:powderblue;">
    • just one part
      < p style="background-color:blue;">
    • use image as background
      < p style="background-image: url("")">
  3. change font size
    < div style="font-size:120%" >
  4. change font
    <p style="font-family:Comic Sans MS;">
  5. make it look
    < b>bold< /b>
  6. make it look
    < i>italic< /i>
