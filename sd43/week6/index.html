to select based on element

This is the problem that we had last class, where change one style for < p> on the top would change all the styles for < p>.

.class

We use .class to give each < > seperate names so the HTML file and recognize it. Since class is assigning same properties to a group under the same name.
*There should be no spaces in the class name, due to space means other things in CSS*
