more attribute on < img>

Images are defined with the < img >
The source file (URL), alternative text (will show note when it is not working), width, and height are provided as attributes

HTML5 Audio

The HTML5 < audio> element specifies a standard way to embed audio in a web page. It has many built in properties, we are going to try autoplay, and loop. First, we need a sound file.

<audio id="myAudio" controls>
<source id="myAudio" src="bgSound.wav" type="audio/wav">
Your browser does not support the audio element.
</audio>

<script>
var vid = document.getElementById("myAudio");
vid.autoplay = true;
vid.loop = true;
</script>
