Key points

styling in HTML

  1. change font color
    <p style="color:red;">
  2. change background color
    • as a whole
      < body style = "background-color:powderblue;">
    • just one part
      < p style="background-color:blue;">
    • use image as background
      < p style="background-image: url("")">
  3. change font size
    < div style="font-size:120%" >
  4. change font
    <p style="font-family:Comic Sans MS;">
  5. make it look
    < b>bold< /b>
  6. make it look
    < i>italic< /i>
lists

2 type of list, orderd list and unordered List

  1. orderd list
      < ol>
    1. < li> item< /li>
    2. < li> item< /li>
      3. < /ol>
  2. unordered List
      < ul>
    • < li> item< /li>
    • < li> item< /li>
      • < /ul>

more attribute on < img>

Images are defined with the < img >
The source file (URL), alternative text (will show note when it is not working), width, and height are provided as attributes

< img src="pokemon.jpg" alt="pokemon not working" width="490" height="245">

< img src="http://www.pokemongo.com/static/assets/images/pokemon_go_logo.png" alt="pokemon not working" width="50%" height="300">
