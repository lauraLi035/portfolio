.additional styling properties

add styles to borders, to add space around a display tag, and set width and height to a tag, display muliple tiles on a row.

  1. margin property has four values:
    • margin: 25px 50px 75px 100px; as top, right, bottom and left
    • auto to center a image,
    • length - specifies a margin in px, pt, cm, etc.
    • % - specifies a margin in % of the width of the containing element
  2. CSS Border Properties
    • p.dotted {border-style: dotted;}
    • p.dashed {border-style: dashed;}
    • p.solid {border-style: solid;}
    • p {
      border-left: 6px solid red;
      background-color: lightgrey;
      }
  3. Setting height and width
    div {
    height: 200px;
    width: 50%;
    background-color: powderblue;}
