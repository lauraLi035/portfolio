Introduction to the Internet and Web Development

In this class we are going to learn how to design and develop dynamic website, through a hands on learning progress. Main Topics are HTML, CSS, JavaScript and based on the course process and studentsâ€™ interest we will introduce Python or Bootstrap. The goal of this class to give fundamental concepts for front end development while able to build a portfolio or blog by the end of the term.
